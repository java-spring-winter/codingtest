### 백준 2438번 별 찍기-1 - https://www.acmicpc.net/problem/2438
```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();

        for(int i = 0; i < n; i++) {
            for(int j = 0; j < i + 1; j++)
                System.out.print("*");
            System.out.print("\n");
        }

        sc.close();
    }
}
```
- 설명
 * for문을 이용하여 입력된 값에 대해 순환해서 * 출력
